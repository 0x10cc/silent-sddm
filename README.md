# Installing
### Dependencies
- SDDM 0.20 or higher;
- qt5-svg;
- qt5-quickcontrols2;
- qt5-graphicaleffects;
- RedHatDisplay font (opcional)

Install dependencies on Arch Linux:
```bash
$ sudo pacman -S sddm qt5-svg qt5-quickcontrols2 qt5-graphicaleffects
$ yay -S redhat-fonts
```
